SNU HTML5 documentation//Definitions//What is HTML5

Definition

HTML (HyperText Markup Language) is the language of an internet page. It is not a programming language, it is a markup language. HTML5 was made in 2012 as a way to have an updated HTML configuration, and to have media in the browser without the need for Adobe Flash.

Web browser support

All web browsers support HTML, only modern browsers support HTML5
