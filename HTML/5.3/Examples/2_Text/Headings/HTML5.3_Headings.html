Description

The heading tag is one of the first HTML tags, and the most common tag. It was introduced in HTML1 in 1993.

The heading tag creates a formatted heading. There are 6 heading types, h1 being the largest, and h6 being the smallest, unless you were to modify them in a CSS spreadsheet.
